Google Sheets — How to Import
  1. Download the Excel (.xlsx) or CSV file above.
  2. Open sheets.google.com and create a new spreadsheet.
  3. Go to File → Import → Upload and select the downloaded file.
  4. Choose "Replace spreadsheet" or "Insert new sheet(s)" and click Import data.
  5. All project details, phases, status values, and notes will be preserved.
